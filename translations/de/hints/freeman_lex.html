Haben Sie eine Idee 💡! Sie müssen eine Folge von Zeichenpaaren mit demselben Index bilden und nur Paare mit unterschiedlichen Zeichen behalten. Es darf nur zwei palindromische Paare geben, um eine Schalterkette zu bilden.

Ok, womit soll ich anfangen?

Gemäß der Beschreibung werden gleiche Saiten als One-Switch-Saiten betrachtet. Sie werden auch feststellen, dass sich die Länge der Zeichenketten durch die Umschaltung nicht ändert. Wenn die angegebenen Zeichenketten also unterschiedlich lang sind, handelt es sich definitiv nicht um eine einzige Umschaltzeichenkette. Lassen Sie uns das mit der Anweisung if korrigieren. 

if line == result:
    return True
if len(line)!= len(result):
    return False
Sie schneidet einige Fälle ab! Was ist als nächstes zu tun?

Jetzt müssen Sie eine Folge von Zeichenpaaren mit demselben Index erstellen. Dafür gibt es eine eigene eingebaute Funktion: zip(). Iterieren Sie über die Sequenz mit der Schleife for und weisen Sie dabei jedes Element des Paares einer anderen Variablen zu. 

for a, b in zip(line, result)

Aus dieser Sequenz müssen Sie nur die Paare behalten, bei denen die Elemente nicht gleich sind. Da es sich bei beiden Elementen um Strings handelt, können wir sie auch zu einem einzigen String verketten und diesen Ausdruck als list comprehension verwenden. 

diff = [a + b for a, b in zip(line, result) if a!= b]
Helfen Sie mir, die Mission zu beenden.

Lassen Sie uns darüber nachdenken, was diff im Falle eines Switch-Strings sein muss. Wenn es sich nur um einen Schalter handelt, dann darf diff nur zwei Zeichenpaare haben und diese Paare müssen gleich sein, aber in umgekehrter Reihenfolge: Wenn an einer Stelle der Zeichenfolge "b" statt "a" steht (verkettetes Paar "ba"), dann steht an einer anderen Stelle "a" statt "b". Lassen Sie es uns folgendermaßen schreiben. 

return len(diff) == 2 and diff[0] == diff[1][::-1]
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise zusammenführen und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def switch_strings(line: str, result: str) -> bool:

    if line == result:
        return True
    if len(line)!= len(result):
        return False
    
    diff = [a + b for a, b in zip(line, result) if a!= b]
        
    return len(diff) == 2 and diff[0] == diff[1][::-1]
